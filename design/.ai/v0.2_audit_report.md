## 1. 앱 목적 및 v0.2 역할

RunPulse는 다중 러닝 플랫폼(Garmin, Strava, Intervals.icu, Runalyze)의 데이터를
SQLite DB로 통합하고, 20개 이상의 1차/2차 가공 메트릭을 계산하여 웹 대시보드로
제공하는 통합 러닝 분석/코칭 플랫폼이다.

비전 문서(통합 UI 설계 연구)는 4개 Phase로 전체 로드맵을 정의한다:

- Phase 1 (0-3mo): 기본 인프라 + 1차 메트릭
- Phase 2 (3-6mo): 2차 메트릭 + AI 코칭
- Phase 3 (6-9mo): 훈련 계획 + 레이스 예측
- Phase 4 (9-12mo): 고도화 + ML 통합

v0.2는 Phase 1 완료 + Phase 2 전반을 커버한다.
핵심 목표: 2차 메트릭 계산 엔진 완성 + 핵심 화면 UI 구현.
