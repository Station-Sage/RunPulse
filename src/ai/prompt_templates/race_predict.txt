## 요청 사항

다음 항목을 포함하여 레이스 준비도 분석 리포트를 작성해 주세요:

1. **현재 피트니스 수준** — CTL, VO2Max(VDOT), Marathon Shape를 종합한 현재 실력 평가
2. **예상 레이스 기록** — 5K / 10K / 하프마라톤 / 풀마라톤 예상 기록
3. **목표 달성 가능성** — 목표 기록(있는 경우) 달성을 위한 현실적 평가
4. **TSB 상태** — 레이스일 기준 신선도(TSB) 예측
5. **남은 기간 훈련 방향** — 레이스까지 남은 기간에 집중해야 할 훈련 요소
6. **레이스 전략** — 목표 페이스 배분 및 레이스 당일 조언

마지막에 후속 질문 3개를 JSON으로 포함해 주세요:
```json
{
  "suggestions": [
    "후속 질문 1",
    "후속 질문 2",
    "후속 질문 3"
  ]
}
```
