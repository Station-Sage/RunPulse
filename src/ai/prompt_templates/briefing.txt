## 요청 사항

위 데이터를 분석하여 다음 항목을 포함한 브리핑을 한국어로 작성해 주세요:

1. **오늘 컨디션 종합** — 회복 점수, Body Battery, HRV, 수면을 종합하여 오늘 훈련 가능 여부와 강도 권장
2. **이번 주 훈련 평가** — 볼륨/강도/일관성 및 계획 이행률(완료/건너뜀 횟수) 평가
3. **주목할 점** — 부상 위험(ACWR), 과훈련/저훈련 신호, 특이사항
4. **오늘 권장 훈련** — 아래를 종합하여 구체적 조언:
   - 어제 훈련 이행 여부 (완료했으면 오늘 볼륨 유지, 건너뜀이면 보상 훈련 필요 여부 판단)
   - 이번 주 누적 이행률이 낮으면 오늘 강도 조정 또는 우선순위 재설정
   - 오늘 계획된 훈련 종류/거리와 현재 컨디션의 적합성
5. **이번 주 나머지 계획** — 건너뜀/이행률 이력을 반영한 잔여 훈련 방향 제안
   - 건너뛴 고강도 훈련이 있으면 이동/보상/생략 중 권장 사항 명시

브리핑 마지막에 아래 JSON 형식으로 추천 후속 질문 3개를 포함해 주세요:
```json
{
  "suggestions": [
    "후속 질문 1",
    "후속 질문 2",
    "후속 질문 3"
  ]
}
```
