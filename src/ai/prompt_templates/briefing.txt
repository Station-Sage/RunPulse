## 요청 사항

위 데이터를 분석하여 다음 항목을 포함한 브리핑을 한국어로 작성해 주세요:

1. **오늘 컨디션 종합** — 회복 점수, Body Battery, HRV, 수면을 종합하여 오늘 훈련 가능 여부와 강도 권장
2. **이번 주 훈련 평가** — 볼륨/강도/일관성 측면에서 이번 주 훈련의 질 평가
3. **주목할 점** — 부상 위험(ACWR), 과훈련/저훈련 신호, 특이사항
4. **오늘 권장 훈련** — 오늘 계획과 컨디션을 고려한 구체적인 훈련 조언
5. **이번 주 나머지 계획** — 남은 요일 훈련 방향 간략 제안

브리핑 마지막에 아래 JSON 형식으로 추천 후속 질문 3개를 포함해 주세요:
```json
{
  "suggestions": [
    "후속 질문 1",
    "후속 질문 2",
    "후속 질문 3"
  ]
}
```
